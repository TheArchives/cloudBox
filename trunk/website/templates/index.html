What is The Archives?

It's a custom Minecraft server which contains historical copies of as many other Minecraft servers as I can get permission for. You can access it here.

Why?

Sandbox mode has led to a lot of interesting building of random things, and some very elaborate construction, and I'm the sort of person who doesn't want to let that go to waste, even if nobody will ever want to see it again.

So, I built a custom server (actually, that part was just for fun) and an archivist client that spiders servers and grabs their maps. Then, they're loaded into my server's archives list, and available for players to load and have a look around.

If you're a server admin and want your server archived, post a link to it in this thread.

How?

Log into the server, and try the following commands: 

                /aname reddit
                /atime newest
                /aboot
That will load up the newest archive of the reddit public server, and load you into it. The server can support multiple 'worlds' at once; it has a very verbose /help command that will assist you with most things, by typing /help commandname (/help brings up a list of commands).
